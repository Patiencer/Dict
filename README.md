##  Key command:
* ###  Build install package
> `npm run dist` 
* ###  Publish to GitHub
>0. set the GH_TOKEN `[Environment]::SetEnvironmentVariable("GH_TOKEN","xxxx","User")`
>1. change version
>2. cd to application directory and run `npm run publish`

* ###  Check the log
> C:\Users\\`UserName`\AppData\Roaming\dict\log.log
